    Self Driving Car Using Javascript - Neural Network Simulation

    Javascript

    A self-driving car simulation using JavaScript typically involves implementing a neural network to make the car learn and navigate a virtual environment without human intervention. Simulate acceleration, braking, and steering. Implement friction and collision detection. Use keyboard controls for manual testing before automation. Road & Environment Define lanes and road boundaries using HTML5 Canvas. Generate obstacles, traffic, or other cars dynamically.     Waste Management Using AIOT

    By the use of Python, Arduino, L293D motor

    Python

    This project introduces a Smart Waste Segregation System that provides an automated solution for garbage sorting and management. The system uses sensors such as moisture sensors and metal sensors to classify waste into three categories: wet waste, dry waste, and metal waste. A conveyor belt mechanism, powered by DC motors, automates the movement of waste materials for segregation. Once the waste is detected and identified by the sensors, it is directed into one of three bins using a slot mechanism driven by a DC motor. The system also incorporates ultrasonic sensors to monitor the garbage levels in each bin. When a bin reaches its maximum capacity, real-time notifications are sent to the authorities through IoT connectivity, ensuring timely clearance. By implementing such technology, this project aims to improve waste management efficiency and promote cleaner environmental conditions.
